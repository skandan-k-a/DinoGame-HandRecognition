A Brief Intro

Our Project can be used to play Chrome's Dinosaur Game with the Help of Gestures. We Used Python Libraries Like Open CV to take in the Gesture Input. Close fist can be used to crouch and open palm to Jump the Cactus in the Game.This ensures Non-Physical Contact with the KeyBoard Keys for the Gameplay.

This Game can also be used to exercise the palms in a Unique Fun way!!
